# Badminton singles matchmaking

A tool that automatically tracks players' elo and handles matchmaking.
To use on Windows, simple unpack and run the executable found at https://github.com/potat-code/Badminton-singles-matchmaking/releases
I might add support for other operating systems too

## ! NOTE: This is in ALPHA !
Some features have not been implemented yet. There are a few thing i'd like to add in future versions:

	Global leaderboard
	Better search filterring
	Settings
	Some sort of audio
